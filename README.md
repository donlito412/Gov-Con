# GovCon Command Center Dashboard

A premium, interactive dashboard for daily SLED and AI contract monitoring.

## Features
- **15 Categorized Sections**: Full coverage of all procurement platforms, SLED opportunities, AI contracts, and supplier networks.
- **Interactive Tables**: Every opportunity and platform is listed with direct source links.
- **Premium Design**: Dark-mode glassmorphism interface for easy daily use.
- **Capture Strategy**: Embedded business model and capture search strings.

## Local Setup
1. Open terminal in this directory.
2. Run `npm install` (already performed by AI).
3. Run `npm run dev` to launch the dashboard.
4. Open the provided local URL in your browser.

## Deployment
This project is initialized and ready to be pushed to GitHub:
```bash
git push -u origin main
```
